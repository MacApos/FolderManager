# FolderManager

This is a simple project for searching and viewing folders in your cabinet.

## Technologies

* Java 21
* JUnit Jupiter 5.8.1
* Java Hamcrest: 3.0
* Apache Maven 3.9.11

## Installation

To clone repository run:

```bash
git clone https://github.com/MacApos/FolderManager.git
```

## Usage

The project's *main()* function showcases all important functions.
To call it, first compile the project with:

```bash
mvn clean package
```
Than run:

```bash
java -jar target/FolderManager-1.0-SNAPSHOT.jar
```
